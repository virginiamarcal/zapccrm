## Field Reference

| Placeholder | Source | Description |
|-------------|--------|-------------|
| `{CYCLE_NUMBER}` | catalog.evaluationCycle | Current cycle |
| `{TIMESTAMP}` | ISO-8601 | Generation time |
| `{SCORING_RULES_VERSION}` | scoring-rules.version | Rules version |
| `{TOTAL_VARIANTS}` | inventory.totalVariants | Count from inventory |
| `{NEW_VARIANTS}` | inventory.newVariants | New since last cycle |
| `{TRANSITIONS_COUNT}` | decisions.length | Auto-transitions |
| `{PROPOSALS_COUNT}` | proposals.length | Pending reviews |
| `{AVG_SCORE}` | Calculated | Mean of all total scores |
| `{MAX_SCORE}` | Calculated | Highest total score |
| `{MIN_SCORE}` | Calculated | Lowest non-REJECT score |

---

## Usage

This template is consumed by `ds-critical-eye-report` task (Step 6 of the Critical Eye workflow). It should NOT be filled manually — the task reads all cycle data files and renders this template automatically.

To generate manually (emergency): read all cycle JSON files from `outputs/design-system/critical-eye/cycle-{N}/` and fill placeholders.
