**Status:** Ready

## Description

Como um **administrador de sistema**,
Quero **migrar o schema do Supabase para suportar múltiplas organizações**,
Para que **cada empresa tenha seus dados completamente isolados**.

---

## Acceptance Criteria

- [x] **AC1:** Tabela `organizations` criada com campos: `id`, `name`, `slug`, `created_at`, `owner_id`
- [x] **AC2:** Todas as 38 tabelas atualizadas com coluna `organization_id` (not null)
- [x] **AC3:** Migration Supabase criada e testada em staging
- [x] **AC4:** Rollback plan documentado (migration reversa)
- [ ] **AC5:** Dados existentes migrados com `organization_id = uuid_nil` (super admin org)
- [ ] **AC6:** Nenhum erro de constraint durante migration
