# Story 3.1: Layout Inbox + Sidebar Contexto

**Status:** Ready
**Sprint:** 5
**Epic:** Epic 3 - Inbox Unificado
**Story ID:** 3.1
**Complexity:** 10 pts

---

## Description

Como um **atendente**,
Quero **ver conversa WhatsApp + card de contato + card de deal na mesma tela**,
Para que **tenha todo contexto sem trocar de aba**.

---

## Acceptance Criteria

- [ ] **AC1:** Layout 3 colunas (60% + 40%): Coluna 1 lista de conversas, Coluna 2 conversa selecionada, Coluna 3 sidebar com contexto
- [ ] **AC2:** Contact card mostra: nome, phone, email, tags, total mensagens
- [ ] **AC3:** Deal card mostra: nome, stage, valor, probability, last update
- [ ] **AC4:** Activity feed: últimas 5 atividades (mensagens, calls, tasks)
- [ ] **AC5:** Sidebar scrollable (overflow: auto)
- [ ] **AC6:** Mobile: stack vertical (inbox → conversa → sidebar em abas)
- [ ] **AC7:** Realtime: nova mensagem adiciona à lista, atividade aparece no feed sem refresh

---

## Scope

### In Scope
- 3-column layout (desktop)
- Contact card component
- Deal card component
- Activity feed component
- Mobile responsive stack layout
- Realtime updates via Supabase

### Out of Scope
- Quick actions (Story 3.2)
- AI suggestions (Story 3.4)
- Message styling/formatting (Epic 4)

## Notes for Developer

- Use Tailwind CSS grid (60% / 40% split)
- Contact card: fetch from contacts table (org_id + contact_id)
- Deal card: fetch from deals table (linked via contact_id or explicit deal selection)
- Activity feed: query activities table, limit to 5 (load more button)
- Realtime: use Supabase `.on('*')` listeners on conversations, activities
- Mobile: use responsive classes (md: grid, sm: flex column)
- Performance: lazy load deal card (only if contact has deals)
