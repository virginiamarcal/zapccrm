# Story 2.4: Deal Creation via WhatsApp (Keyword Trigger)

**Status:** Ready
**Sprint:** 4
**Epic:** Epic 2 - Integração WhatsApp↔CRM
**Story ID:** 2.4
**Complexity:** 8 pts

---

## Description

Como um **SDR**,
Quero **que uma mensagem WhatsApp com keyword específica crie automaticamente um deal**,
Para que **não perca leads qualificados**.

---

## Acceptance Criteria

- [x] **AC1:** Admin pode configurar keywords por org (ex: "quero orçamento", "preciso de demo")
- [x] **AC2:** Função PL/pgSQL `process_inbound_message` verifica keywords
- [x] **AC3:** Se match: cria Deal com: `contact_id`, `board_id` (padrão "Web Leads"), `stage_id` (primeiro stage "Novo"), `name` (auto-gerado), `deal_value` (default null)
- [x] **AC4:** Deal criado dispara notificação para assign (Slack, email)
- [x] **AC5:** Testes: mensagem com keyword cria deal, sem keyword não cria

---

## Notes for Developer

- Keywords table: org_id, keyword (text), board_id, stage_id
- Matching: case-insensitive, partial match (use `ILIKE` in SQL)
- Deal name template: "Lead from WhatsApp - {contact_name} - {date}"
- Notification: emit event that Epic 5 (Notifications) can consume
- PL/pgSQL function: called from webhook handler (Story 2.2)
- Performance: cache keywords in memory (TTL 5min) to avoid DB hit per message
