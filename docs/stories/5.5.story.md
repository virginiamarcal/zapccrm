# Story 5.5: Checkout e Onboarding com Upgrade

**Status:** Draft | **Sprint:** 10 | **Complexity:** 8 pts

Como um **usuario**, Quero **passar do trial para pagante sem perder dados**, Para que **a transição seja suave**.

**AC:**
- [ ] Button "Upgrade Now" em `/settings/billing` ou dashboard
- [ ] Clica → redireciona para `/api/billing/create-checkout-session`
- [ ] Stripe Checkout carrega (card, email, billing address)
- [ ] Webhook processa `payment_intent.succeeded`
- [ ] Organization marcada `trial_status = 'converted'`, `status = 'active'`
- [ ] Email enviado: "Obrigado! Seu subscription foi ativado"
