# Story 2.3: Conversa WhatsApp → Activity CRM (Timeline Unificada)

**Status:** Ready
**Sprint:** 3-4
**Epic:** Epic 2 - Integração WhatsApp↔CRM
**Story ID:** 2.3
**Complexity:** 8 pts

---

## Description

Como um **gerente**,
Quero **ver todas as interações (WhatsApp, email, ligações) no histórico do contato**,
Para que **tenha visão 360° do cliente**.

---

## Acceptance Criteria

- [ ] **AC1:** Cada mensagem WhatsApp cria Activity com: `activity_type = 'whatsapp_message'`, `contact_id`, `content`, `direction` (inbound/outbound), `created_at`
- [ ] **AC2:** Activity visível em timeline do contact (ordenado por created_at)
- [ ] **AC3:** Contadores atualizados (total mensagens, status lido/não lido)
- [ ] **AC4:** Testes: 10 mensagens WhatsApp geram 10 activities CRM

---

## Scope

### In Scope

- Message counters
- Tests for activity flow

## Notes for Developer

- Activity creation is IMMEDIATE (on webhook receipt)
- Direction: "inbound" (from customer), "outbound" (from agent)
- Content: plain text of message (strip media URLs for now)
- Counters: cache in activities table or compute on-demand (optimize later)
- Read status: message level flag (WhatsApp tracks this, sync to activity)
- Ordering: DESC by created_at (newest first)
