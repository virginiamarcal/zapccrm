# Story 2.2: Contato WhatsApp → Contact CRM (Auto-Criação + Merge)

**Status:** Ready
**Sprint:** 3
**Epic:** Epic 2 - Integração WhatsApp↔CRM
**Story ID:** 2.2
**Complexity:** 8 pts

---

## Description

Como um **agente de vendas**,
Quero **que contatos WhatsApp apareçam automaticamente no CRM**,
Para que **eu tenha histórico completo sem criar manualmente**.

---

## Acceptance Criteria

- [x] **AC1:** Webhook WhatsApp recebe novo contato (`from` field)
- [x] **AC2:** Verifica se contact existe (phone E.164 match ou email)
- [x] **AC3:** Se não existe: cria contact com nome (se disponível), phone, inicializa org_id
- [x] **AC4:** Se existe: merge — atualiza last_contact_date, adiciona tags `whatsapp_active`
- [x] **AC5:** Contato criado dispara evento `contact.created` para listeners
- [x] **AC6:** Testes: novo contato da WhatsApp cria contact CRM, contato existente merges corretamente
