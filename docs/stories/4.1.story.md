# Story 4.1: Deal Stage → Template WhatsApp Auto-Send

**Status:** Draft
**Sprint:** 7
**Epic:** Epic 4 - Automações Cross-Platform
**Story ID:** 4.1
**Complexity:** 8 pts

---

## Description

Como um **gerente**,
Quero **configurar que quando um deal muda para "Proposta Enviada", uma mensagem automática é enviada**,
Para que **o cliente sempre saiba que recebeu a proposta**.

---

## Acceptance Criteria

- [ ] **AC1:** Page `/flows` mostra builder visual com nodes: Start → Trigger → Message → End
- [ ] **AC2:** Admin seleciona stage ("Proposta Enviada")
- [ ] **AC3:** Admin seleciona template (com preview)
- [ ] **AC4:** Admin clica "Deploy" → cria workflow record
- [ ] **AC5:** Função PL/pgSQL dispara quando deal muda de stage
- [ ] **AC6:** Testes: move deal → mensagem enviada com 3s latência
