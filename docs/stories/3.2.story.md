# Story 3.2: Quick Actions na Conversa

**Status:** Ready
**Sprint:** 5
**Epic:** Epic 3 - Inbox Unificado
**Story ID:** 3.2
**Complexity:** 8 pts

---

## Description

Como um **vendedor em movimento**,
Quero **enviar resposta, transferir conversa ou criar tarefa sem deixar inbox**,
Para que **seja rápido atender**.

---

## Acceptance Criteria

- [ ] **AC1:** Botões flutuantes em hover da conversa: "Reply", "Assign", "Create Task", "Mark as Done"
- [ ] **AC2:** Reply auto-insere o saudação (ex: "Oi [nome]")
- [ ] **AC3:** Assign → dropdown de team members
- [ ] **AC4:** Create Task → modal de task (due date, assignee, description)
- [ ] **AC5:** Mark as Done → move para done, envia feedback message
- [ ] **AC6:** Testes: clica Reply, escreve, envia → mensagem em WhatsApp em < 1s

---

## Scope

### In Scope
- Quick action buttons (Reply, Assign, Create Task, Mark Done)
- Reply input with auto-greeting
- Assign dropdown with team members
- Task creation modal
- Done status transition
- Sub-1s send latency

### Out of Scope
- Bulk actions (multi-select)
- Custom quick replies (Epic 4)
- Task due date notifications (future)

## Notes for Developer

- Buttons appear on hover (not always visible, save space)
- Reply greeting: "Oi {{contact_name}}" (fetch from contact card)
- Assign dropdown: fetch team members from org_members table
- Task modal: use React Hook Form + modal component (shadcn)
- Mark Done: update conversation status, optionally send auto-response
- Send latency: use QStash (durable steps) for reliability
- Performance: debounce hover events (prevent flickering)
