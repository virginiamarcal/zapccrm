# Story 4.4: AI Decisions do CRM → Auto-Send Mensagem

**Status:** Draft
**Sprint:** 8
**Epic:** Epic 4 - Automações Cross-Platform
**Story ID:** 4.4
**Complexity:** 10 pts

---

## Description

Como um **diretor de vendas**,
Quero **que se AI sugere "contato está pronto para upsell", uma oferta seja enviada via WhatsApp automaticamente**,
Para que **maximize conversão**.

---

## Acceptance Criteria

- [ ] **AC1:** AI Decisions geradas por análise: recency, value, comportamento
- [ ] **AC2:** Decision tem `status = 'suggested'`, `confidence_score` (0-100)
- [ ] **AC3:** Admin configura ação para cada decision type: Decision "upsell_ready" → Action "Send upsell_template"
- [ ] **AC4:** Quando decision criada: função PL/pgSQL verifica ação, enqueue no QStash
- [ ] **AC5:** Dashboard mostra "Decisions Acted" vs "Decisions Ignored" (metrics)
- [ ] **AC6:** Testes: AI decision criada → mensagem enviada em < 5s

---

## Scope

### In Scope
- AI decision generation (scoring engine)
- Decision storage with confidence
- Action mapping per decision type
- Automatic action execution
- Metrics dashboard (acted vs ignored)
- Tests for latency < 5s

### Out of Scope
- ML model training (future)
- Custom decision types (future)
- Feedback loop for model improvement (future)
