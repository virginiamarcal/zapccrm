## Notes for Developer

- Use role-based checks: `SELECT (...WHERE auth_role() IN ('admin', 'owner'))`
- Test edge cases: service_role bypass, deleted users, role changes
- Audit log EVERY write operation (compliance requirement)
- Performance: RLS adds minimal overhead (< 2ms per query)
