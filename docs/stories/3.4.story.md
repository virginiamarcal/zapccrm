**Status:** Ready
**Sprint:** 6

---

## Description

Como um **atendente**,
Quero **que o sistema sugira respostas baseadas no histórico do deal**,
Para que **responda mais rápido e consistente**.

---

## Acceptance Criteria

- [ ] **AC1:** Input de resposta mostra "💡 Suggestions" com 2-3 opções de resposta
- [ ] **AC2:** Suggester usa: Contexto do deal (stage, valor, histórico), Histórico de mensagens recentes, Templates pré-configurados, LLM (Anthropic/OpenAI)
- [ ] **AC3:** Clica em suggestion → preenchе input (ainda edita antes de enviar)
- [ ] **AC4:** Log de qual suggestion foi usada (metrics)
- [ ] **AC5:** Testes: sugestão relevante ao contexto do deal, está pronta em < 2s

## Notes for Developer

- LLM: use Anthropic Claude API (claude-3-sonnet for speed)
- Prompt: "Given deal context [stage, value, history], suggest 2-3 professional responses"
- Context: include deal stage, last message, contact name, time of day
- Caching: cache suggestions per conversation (TTL 5min)
- Fallback: if LLM fails, show template suggestions only
- Usage tracking: log suggestion_id + whether it was used
- Performance: use async fetch (don't block input)
- Security: sanitize LLM output (no XSS)
